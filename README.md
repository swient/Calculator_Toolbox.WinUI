# Calculator Toolbox

</div>

## 簡介

Calculator Toolbox 是一個基於 WinUI 3 的簡易計算機工具箱

## 功能

- [x] 計算機
- [ ] 進制轉換
- [ ] 單位換算

## 安裝與執行

1. 需安裝 [.NET 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) 或以上版本
2. 使用 Visual Studio 開啟 `Calculator_Toolbox.WinUI.slnx`
3. 建置並執行專案，即可使用計算機工具箱

## 專案結構

- `CalculatorToolbox/MainWindow.xaml.cs`：主視窗程式邏輯
- `CalculatorToolbox/CalculatorView.xaml.cs`：計算機程式邏輯
- `CalculatorToolbox/CalculatorEngine.cs`：運算邏輯
- `CalculatorToolbox/App.xaml.cs`：程式進入點
- `CalculatorToolbox/CalculatorToolbox.csproj`：專案設定
